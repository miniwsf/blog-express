# blog-express

#### 介绍
使用express搭建的个人博客

#### 目录结构
|--controller  控制器文件
|--middlewares  中间件（登录验证）
|--models      数据模型
|--mongodb     mongodb数据库连接
|--public      静态文件
|--routes      路由
|--dist        编译后资源文件
|--src         主要源码
|--|--common   公共组件（eg：弹框等）
|--|--css      与模块相关的css文件
|--|--images   与模块相关的image
|--|--js       与模块相关的js
|--|--views    具体页面
